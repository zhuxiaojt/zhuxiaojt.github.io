# Isotopic 1024 中文版
翻译自[Isotopic 1024](https://github.com/glukascewic1/Isotopic2048/)，并改进手机适配。

[2048](https://github.com/gabrielecirulli/2048)的一个小克隆体，一个基于[Saming 的 2048](http://saming.fr/p/2048/)（同样是克隆体）的[1024](https://play.google.com/store/apps/details?id=com.veewo.a1024)小克隆体。

将元素连接起来，找到质量为256的地块！不过要小心不稳定的元素，因为它们不会永远存在！
